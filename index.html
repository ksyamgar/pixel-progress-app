This is the Pixel Progress Next.js Application

If you are seeing this index.html page when deploying to Netlify, it might mean that your Netlify site is not correctly configured to build and serve this as a Next.js project.

For Next.js applications, Netlify typically handles the build process (e.g., running npm run build or next build) and serves the app from the generated .next directory automatically. A root index.html file like this one is usually not needed or displayed when Netlify is correctly set up for Next.js.

Please ensure your Netlify deployment settings are configured for a Next.js application:

  • Build command: npm run build (or next build)
  • Publish directory: .next

Your actual application should be accessible once these settings are correct.
